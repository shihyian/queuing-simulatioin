# queuing-simulatioin
在不同arrival和service rate，及inter arrival rate和process time變異程度下模擬排隊情形，介面包含control panel, svg animation及動態圖表。
使用者可設定：
1. 排隊人數
2. arrival rate, service rate, arrival and service cv(coefficient of variation)
3. 動畫速度
